## 🎓 USAGE GUIDE

### Command Palette
Press `Ctrl+K` (Windows) or `Cmd+K` (Mac) anywhere in CRM to:
- Quick create entities
- Navigate to pages
- Search commands

### Inline Editing
Click any field with edit icon to:
- Edit in place
- Press Enter to save
- Press Escape to cancel

### Email Templates
Navigate to Settings → Email Templates:
- Create reusable templates
- Use variables for personalization
- Categorize by type

### Document Uploads
On Contract/Invoice detail pages:
- Click upload area
- Select PDF/DOC file
- Download or delete anytime

### HubSpot Sync
Navigate to Settings → HubSpot Sync:
- Monitor sync status
- Retry failed syncs
- View sync statistics

### Drag-Drop Pipeline
On Deals page:
- Drag deals between stages
- Auto-updates probability
- Visual stage indicators

### @Mentions
In any note/comment field:
- Type `@` to mention users
- Select from dropdown
- Users get notified

### Workflow Automation
Navigate to Settings → Workflows:
- Create automated workflows
- Add triggers and conditions
- Define actions

### Saved Filters
On any list page with filters:
- Apply filters
- Click "Save Filter"
- Load presets anytime
